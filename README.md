# Orangex
中 文 编 程 ， 最 为 致 命

# 使用
安装:  
```sh
npm install orangex -g  
橙式  
```
帮助:  
```sh
橙式 帮助  
```
将当前命令行所在目录下的所有英文文件转换成中文, 并加上.orz后缀后保存:  
```sh
# 此处为浅层转换, 如要遍历子目录就执行  
# 橙式 英转汉 -深  
橙式 英转汉  
```
将当前命令行所在目录下的所有中文文件转换成英文, 并去掉.orz后缀后保存:  
```sh
# 此处为浅层转换, 如要遍历子目录就执行  
# 橙式 英转汉 -深  
橙式 英转汉  
```

# 编译
```sh
git clone https://github.com/OrangeX4/Orangex.git --depth=1  
cd Orangex  
cd nodejs  
npm install  
# 或者 npm test  
npm start  
```
